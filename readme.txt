== Description ==
Sticky Menu allow you to choose any element like header, menus, sidebar etc on your page that you want to sticky at top when page scroll.

== Installation ==

1. Download plugin zip file to your computer and extract in a folder
1. Upload `woc-sticky-menus` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
